# Overtone Singer

Real-time vocal overtone analyzer and spectrogram visualizer.

**Live Demo:** https://remidz.github.io/overtonesinger

## Features

- Real-time audio analysis from your microphone
- Interactive spectrogram visualization
- Adjustable brightness, contrast, and sharpness
- Multiple color schemes
- Record, playback, and export (WAV, PNG, Video)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
